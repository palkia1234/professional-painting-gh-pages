# Professional Painting - Father & Son

Sitio web estático para GitHub Pages de Professional Painting - Father & Son.

## Descripción

Este repositorio contiene el sitio web compilado y listo para producción. El contenido se genera automáticamente desde el repositorio principal del proyecto.

## Cómo usar

1. Clona este repositorio
2. Sirve los archivos estáticos desde un servidor web
3. O configura GitHub Pages para servir directamente desde la rama `main` o `master`

## Configuración en GitHub Pages

1. Ve a la configuración del repositorio en GitHub
2. En la sección "Pages", selecciona la rama donde está este código (generalmente `main` o `master`)
3. Selecciona la carpeta raíz como fuente
4. Guarda los cambios

Tu sitio estará disponible en: `https://<tu-usuario>.github.io/<nombre-repo>/`

## Estructura

- `index.html` - Página principal
- `assets/` - Recursos compilados (CSS y JavaScript)
- Imágenes y otros archivos estáticos

## Actualización

Cuando necesites actualizar el sitio:
1. Regenera el build en el repositorio principal
2. Copia el contenido de `dist/` aquí
3. Commit y push a GitHub
